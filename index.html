Nesta aula, consegui implementar várias funções e gramáticas da linguagem de programação JavaScript. Para evitar repetições na aula, gostaria de personalizar vídeos de forma eficiente, usando apenas um código que pode ser aplicado a todos os vídeos. Para implementar outro vídeo, basta adicionar o seguinte código:

no js


initializeVideoControls('video-element+número_que_você_usar', 
                      'progress-bar+número_que_você_usar', 
                      'btnReplay+número_que_você_usar',
                      'btnPlayPause+número_que_você_usar', 
                      'btnStop+número_que_você_usar', 
                      'volume-bar+número_que_você_vai_mudar', 
                      'btnMute+número_que_você_vai_mudar', 
                      'btnFullScreen+número_que_você_vai_mudar');

no html


 <div id='player+ o numero que vc colocou no js' class="player">
    <video id='video-element+ o numero que vc colocou no js' preload="metadata" lazy="loader" poster="">
        <source src='https://github.com/josemoises2007/alurabooks/assets/106701602/01fc6948-11ee-46c1-869f-450259a16515' type='video/mp4'>
        <source src='https://github.com/josemoises2007/alurabooks/assets/106701602/01fc6948-11ee-46c1-869f-450259a16515' type='video/ogg'>
    </video>
    <div id='controls+ o numero que vc colocou no js' class="controls">
        <progress id='progress-bar+ o numero que vc colocou no js' min='0' max='100' value='0'>0% played</progress>
        <button id='btnReplay+ o numero que vc colocou no js' class='replay' title='replay' accesskey="R" onclick='replayVideo("+ o numero que vc colocou no js");'>Replay</button>    
        <button id='btnPlayPause+ o numero que vc colocou no js' class='play' title='play' accesskey="P" onclick='playPauseVideo("+ o numero que vc colocou no js");'>Play</button>
        <button id='btnStop+ o numero que vc colocou no js' class='stop' title='stop' accesskey="X" onclick='stopVideo("+ o numero que vc colocou no js");'>Stop</button>
        <input type="range" id="volume-bar+ o numero que vc colocou no js" class="volume-bar" title="volume" min="0" max="1" step="0.1" value="1" onchange='updateVolume("+ o numero que vc colocou no js");'>
        <button id='btnMute+ o numero que vc colocou no js' class='mute' title='mute' onclick='muteVolume("+ o numero que vc colocou no js");'>Mute</button>    
        <button id='btnFullScreen+ o numero que vc colocou no js' class='fullscreen' title='toggle full screen' accesskey="T" onclick='toggleFullScreen("+ o numero que vc colocou no js");'>[  ]</button>
    </div>
</div>

Esse código permitirá que você faça as mesmas personalizações nos vídeos, sem a necessidade de repetir o código HTML.
